# Setup Instructions

1. **Update environment variables**

   Create a `.env.local` file using `.env.example` as a template

2. **Install dependencies**

```bash
   npm install
```

3. **Database setup**

```bash
   npm run db:generate
   npm run db:push
```

4. **Run the development server**

```bash
   npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
